# 🚀 Mini CRM – Roadmap

## 🟢 Phase 1 – Core CRM (Priority)

### 1. Lead Detail Page

Route:
`/leads/[id]`

Features:

* Customer info
* Lead metadata
* Notes
* History (future)

---

### 2. Create Lead Page

Route:
`/leads/new`

Features:

* Form input
* Validation
* Submit → API
* Redirect to list

---

### 3. Update Lead Status

Features:

* Click status badge to update
* Quick workflow:

  * NEW → FOLLOW_UP
  * FOLLOW_UP → NEGOTIATING

---
